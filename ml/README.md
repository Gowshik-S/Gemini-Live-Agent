# ML Layer

This folder contains optional machine-learning components for user-pattern and struggle-related modeling.

Primary files:
- `feature_engine.py`: feature extraction pipeline.
- `ensemble_model.py`: model composition logic.
- `user_model_manager.py`: model lifecycle/load/save helpers.
- `train.py`: training entrypoint.

Model/data folders:
- `models/`: serialized model artifacts.
- `datasets/`: dataset notes and scaffolding.
